## 关键结论

BN gamma 是最稳定且成本最低的遗漏语义。它只增加 `4,800` 个参数，即模型参数的 `0.0428%`，却在全部八个 k 上同时降低 accuracy 和 fidelity、提高 KL。最重要的是，`Top-10 + BN gamma` 已达到 `0.1506/0.1551/2.8540`，在三项指标上同时达到或越过当前 soft 黑盒参考线，而保护比例仅为 `9.0362%`。统一初始化后，过去“需要 Top-16 才达到黑盒”的判断不再成立。

`Top-10 + BN gamma + Downsample` 也以 `10.5684%` 的保护比例达到 `0.1502/0.1589/2.8471`。但它比只加 BN gamma 多保护 `172,032` 个参数，三项指标没有形成一致支配，因此不能把全部 downsample 视为达到黑盒的必要闭包。

Downsample Conv 单独增加 `1.5322%` 参数，所有 k 都产生可测保护作用，但没有一个点严格同时越过三条黑盒参考线。例如 Top-12 + Downsample 的 accuracy/fidelity 已到 `0.1500/0.1608`，KL `2.8339` 仍略低于 `2.835290`。它应保留在后续计算图候选中，但当前优先级低于 BN gamma。

BN gamma 与 Downsample 的组合在多数 k 上达到黑盒，不过并非始终优于 BN gamma 单项；Top-15 还出现 fidelity 和 KL 轻微未达线。这说明语义状态的经验保护作用不保证按集合包含关系单调，攻击者优化路径会随公开/victim 状态组合改变。

Stem Conv 的作用较弱且不稳定。`all_extras` 也没有复现单项贡献的简单叠加，多数 k 明显弱于只加入 BN gamma；Top-17 + All extras 仍未严格达到三项黑盒边界。后续通道块方法不应把全部低成本语义无差别并入，而应让计算图规则决定携带哪些通道对应状态。

Top-17 + All extras 的结果为 `0.1552/0.1629/2.770597`，独立 Lab05 `weight` 为 `0.1540/0.1642/2.767021`，差值只有 `+0.0012/-0.0013/+0.003576`。两套语义集合不完全相同，但在统一轨迹下已足够接近，可作为实现一致性的辅助检查。

这些结果表明 BN gamma 是当前最值得纳入通道块节点状态的部分，但尚未证明 TensorShield rank 能泛化到其他模型，也未证明 `Top-10 + BN gamma` 在多随机种子下稳定达到黑盒。下一阶段应保持同一初始化规则，先做多种子复现，再让通道块选择规则与 TensorShield 固定 rank 脱钩。
