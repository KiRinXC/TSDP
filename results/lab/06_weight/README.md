train/validation，并按 validation soft cross-entropy 选择最早的最优 checkpoint；

## 关键结论

BN gamma 是本实验中最稳定、成本最低的遗漏语义。它只增加 4,800 个参数，即
`0.0428%`，却在全部八个 k 上同时降低 accuracy 和 fidelity、提高 posterior KL。
严格按 soft 黑盒三条原始数值同时判断时，`Top-11/12/13/17 + BN gamma` 已越过
参考线；`Top-10 + BN gamma` 的 accuracy 与 fidelity 已越线，但 KL
`3.005191` 仍低于 `3.039817`。

`Top-10 + BN gamma + Downsample` 保护 `10.5684%` 参数，得到
`0.1240/0.1336/3.072730`，三项均越过 soft 黑盒；`Top-12` 的同类组合也越线。
这里“越过”只表示当前固定攻击训练发生了负迁移，不能解释为安全性优于黑盒：攻击者
始终可以退回只使用 query 的 soft 黑盒训练。因此这些点的有效结论是当前暴露状态
未给该攻击器带来可测收益。

Downsample Conv 单独增加 `1.5322%` 参数，在全部 k 上都有一致方向的保护作用，但
没有一个点同时越过三条 soft 黑盒线。Stem Conv 的作用更弱。二者仍应作为后续计算图
中的候选边或闭包状态，而不应无条件加入所有选中通道。

集合包含关系没有带来指标单调性：BN gamma 与 Downsample 的组合并不总优于
BN gamma 单项，`all_extras` 在多个 k 上反而使固定攻击更强。保护 mask 的信息暴露
本身具有单调关系，但有限 query、固定初始化和固定优化器下的 surrogate 训练结果
不保证单调；不能把单次优化负迁移误写成额外状态会降低安全性。

`Top-17 + all_extras` 保护 `99.9572%` 参数，得到
`0.1418/0.1506/2.918726`。Lab05 的全部 weight 为
`0.1435/0.1490/2.902834`；两者差值较小，但前者额外保护分类头 bias，不能视为
完全相同的 mask。

这些结果支持把 BN gamma 纳入通道/filter 节点的联动状态，并保留 downsample
计算图关系；它们尚未证明 TensorShield 固定 rank 可泛化到其他模型，也没有完成
多随机种子统计。
