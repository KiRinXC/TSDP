# 实验 01 结果

本目录保存 `ResNet18+CIFAR-100` 公开特征的无监督 KMeans 结果。

```text
样本数: 10000
特征维度: 512
聚类数: 100
KMeans 重启次数: 10
KMeans inertia: 3604091.75
```

关键原始指标：

```text
Hungarian 一对一匹配准确率: 30.53%
逐 cluster 多数标签准确率: 33.06%
NMI: 0.4901
```

`metrics.json` 保存完整参数、cluster 到类别的映射和指标；两张 PNG 保存 100 类混淆矩阵。聚类过程没有使用 CIFAR-100 标签，标签只在评估阶段参与匹配和 NMI 计算。

## 实验结论

公开 ImageNet 预训练 ResNet18 特征在没有 CIFAR-100 监督训练时，已能得到 `30.53%` 的一对一聚类匹配准确率和 `0.4901` 的 NMI，明显高于 100 类随机对应的水平。这说明公开预训练权重本身保留了可用于 CIFAR-100 的类别结构，为后续比较“保留并适配公开分类能力”和“直接替换分类头”提供了依据。该结果是无监督特征可分性结论，不等同于代理模型的监督分类准确率，也不能单独说明其对 victim 的拟合能力。
